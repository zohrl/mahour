# ماهورا - قالب فروشگاه پوشاک زنانه

این پروژه یک فرانت‌اند آماده برای فروشگاه آنلاین پوشاک زنانه با چیدمان راست‌به‌چپ فارسی است.

## ساختار

- `index.html`: صفحه اصلی
- `shop.html`: لیست محصولات و فیلترها
- `product.html`: جزئیات محصول
- `cart.html`: سبد خرید و پرداخت
- `account.html`: حساب کاربری
- `assets/css/styles.css`: استایل اصلی
- `assets/js/app.js`: رفتارهای رابط کاربری
- `assets/images/`: تصاویر پروژه

## آماده برای اتصال

نام‌گذاری بخش‌ها و فایل‌ها به شکلی نگه داشته شده که بتوان آن را به قالب وردپرس، Blade لاراول یا کامپوننت‌های بک‌اندی تبدیل کرد. داده‌های نمونه در حال حاضر داخل HTML هستند و می‌توانند بعدا با خروجی پایگاه داده جایگزین شوند.
